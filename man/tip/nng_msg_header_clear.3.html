NAME

nng_msg_header_clear - clear message header

SYNOPSIS

#include <nng/nng.h>

void nng_msg_header_clear(nng_msg *msg);

DESCRIPTION

The nng_msg_clear() function resets the header length of msg to zero.

RETURN VALUES

None.

ERRORS

None.
